# Logos moyens de paiement (BABIFIX)

Les **Mobile Money** (Orange, MTN, Wave, Moov) sont servis en **PNG** via `adminpanel.constants.PAYMENT_METHOD_STATIC` et l’API `/api/public/payment-methods/`. Les anciens SVG restent comme secours de dev uniquement.

## Conformité marque

Respectez les **guides d’usage** des marques (marges, fonds autorisés, interdiction de déformation) :

- Orange Money — [Orange CI](https://www.orange.ci)
- MTN MoMo — [MTN](https://www.mtn.com)
- Wave — [Wave CI](https://wave.com)
- Moov Africa — [Moov Africa CI](https://moov-africa.ci)

## Fichiers

| Fichier | Usage |
|---------|--------|
| `especes.svg` | Icône générique espèces (pas de marque) |
| `orange-money.png` | Orange Money (apps Flutter + dashboard) |
| `mtn-momo.png` | MTN Mobile Money |
| `wave.png` | Wave |
| `moov-money.png` | Moov Money |
